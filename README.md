# Dotfiles

> My dotfiles.


## Installation

```bash
git clone --bare git@github.com:monkeymonk/dotfiles.git $HOME/.dotfiles
```

reload terminal then `dotfiles config --local status.showUntrackedFiles no`


## Usage

```bash
dotfiles status
dotfiles add .somefile
dotfiles commit -m "Add somefile"
dotfiles push
```
